# Dots and Boxes

A turn-based multiplayer game like [dots and boxes](https://en.m.wikipedia.org/wiki/Dots_and_Boxes)

## Features

* Fix singleplayer again AI using min-max and alpha-beta pruning
* Maybe some update to play vs multi AI
